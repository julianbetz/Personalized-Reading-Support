yokome.features.tree¶

class yokome.features.tree.AndOrTree¶

A tree node that conjunctively gathers disjunctive children.

Semantically, children of this node represent cooccurrences. They may only be instances of OrAndTree, i.e. they semantically are alternatives.

Nodes of this type do not carry any satellite data.

class yokome.features.tree.DataOnlyTree(value=None)¶

A tree node that conjunctively gathers conjunctive children.

Semantically, children of this node represent cooccurrences. They may only be other instances of DataOnlyTree.

Nodes of this type may carry satellite data.

class yokome.features.tree.OrAndTree(value=None)¶

A tree that disjunctively gathers AndOrTree children.

Semantically, children of this node are alternatives. They may only be instances of AndOrTree, i.e. they semantically represent cooccurrences.

Nodes of this type may carry satellite data.
